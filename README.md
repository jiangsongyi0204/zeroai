# 零基础AI入门教学视频项目
一个面向零基础人群的AI基础教育开源项目。

## 学习AI的痛点
- 1.打开视频，一上来就是满屏没见过的复杂公式，直接劝退。
- 2.零零碎碎学习了一些AI知识，对AI没有一个系统性，结构性的理解。
- 3.跟风潮流，学了一些AI工具。但是没有理解AI的本质原理，无法建设性的运用AI。

## 本项目的目标
- 1.适合零基础的小白人群。
- 2.内容讲解要符合KISS(Keep It Simple, Stupid)原则。
- 3.利用Excel的数据展示能力，图形化，直觉化的说明本质原理，与工程实践。
